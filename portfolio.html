Classificador de Imagens de Animais usando Google Generative AI

Script Python que usa a biblioteca Google Generative AI para classificar animais em imagens. O script recebe um diretório de arquivos de imagem como entrada, classifica os animais em cada imagem usando um modelo gerativo e renomeia os arquivos com base no nome da espécie do animal classificado.

Análise de Dados Sintéticos

Código em Python para gerar dados sintéticos para 9 tabelas, onde cada tabela contém um conjunto de valores y gerados aleatoriamente com base em um valor inicial de 5 e um limite superior de 8. O código também gera um valor x correspondente aos índices dos valores y. Em seguida, o código ajusta uma regressão linear aos dados de cada tabela e armazena os coeficientes de regressão em uma lista. O código também calcula outras estatísticas, como o tamanho da tabela, a média, o desvio padrão e a derivada dos valores y. Por fim, o código gera um gráfico de linhas usando a biblioteca Plotly e um heatmap usando a biblioteca Seaborn para mostrar a correlação entre as estatísticas calculadas.

Organize Files

Organize Files é um script simples que ajuda a organizar arquivos em diferentes pastas com base em seus tipos. Ele usa a biblioteca tkinter para selecionar uma pasta raiz e a biblioteca os para manipular arquivos e pastas.
